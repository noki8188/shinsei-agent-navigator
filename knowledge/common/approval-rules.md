# 共通申請・稟議承認基準

## 1. 適用範囲

本基準は、社員が起票する `経費精算` `備品購入` `出張申請` に適用する。  
正式な決裁は社内ワークフロー上の承認結果を優先し、本書は起票時点の判断目安として扱う。

## 2. 承認レベル

- 5万円未満の定型支出: 所属長承認
- 5万円以上 20万円未満の支出: 所属長、部門長承認
- 20万円以上 50万円未満の支出: 所属長、部門長、管理部門承認
- 50万円以上または契約を伴う支出: 稟議起票を必須とし、所属長、部門長、管理部門、必要に応じて役員承認

## 3. 添付書類の基本ルール

- 経費精算は領収書または電子明細を添付する
- 備品購入は 5万円以上で見積書、20万円以上で比較資料または選定理由を添付する
- 出張申請は旅程、概算費用、出張目的を記載する
- 顧客接待や会食を伴う支出は参加者情報と実施目的を補足する

## 4. 起票時の注意

- 申請の件名は「対象」「目的」「金額」が分かる表現にする
- 費用負担部門またはコストセンターが未確定の場合は、提出前に所属長へ確認する
- 緊急購入や事後申請は理由を補記し、人手確認を前提とする
